#100sitioswebchallenge – El reto de los 100 sitios web

Fecha de publicación abril 15, 2020

El reto de los 100 sitios web consiste en hacer 100 páginas web, cada una con diferentes diseños y con distintos niveles de complejidad.

Me decidí a crear este proyecto durante lo que queda del año 2020, pensando que este proyecto puede ser una oportunidad para retarme y aprender nuevas habilidades.

Considero que tengo un nivel básico de conocimientos de HTML y CSS, y tengo expectativas de que este reto me pueda forzar a aplicar nuevo conocimiento.

Las condiciones para este reto que me impondré serán las siguientes:

1.Crear Plantilla de Diseño

Deberé de crear una plantilla de diseño que mostrará cómo se tiene que ver el sitio web.

2.Crear el Sitio Web

Basada en la plantilla de diseño iniciaré a crear la estructura con HTML y CSS como mínimo, y conforme vaya avanzando agregaré JavaScript.

3.Sitio Web Responsivo

Me aseguraré que el sitio web sea visible en teléfonos, ipads, tablets y otras marcas de móviles. Me basaré en las herramientas de simulación que el navegador Chrome ofrece.

3.Subir sitio final a GitHub

Una vez que tenga listo el código del sitio web y haya hecho pruebas subiré el código a mi cuenta en GitHub.

4.Escribir Experiencia en Blog

Por último compartiré la experiencia que tuve al realizar cada uno de los sitios web. Esperando que pueda ser de ayuda para cualquier persona que vaya iniciando en este campo o como yo, que necesite un buen reto para regresar al campo de desarrollo web.

Sígueme durante este reto de los 100 sitios web a lo largo del año, y descubre conmigo nuevos rincones del diseño web.
